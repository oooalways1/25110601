# 🌧️ 수학 산성비 게임 ⚡

초등학교 3학년 학생들을 위한 곱셈과 나눗셈 학습 게임입니다.

## 📋 프로젝트 소개

수학 산성비 게임은 떨어지는 수학 문제를 풀면서 곱셈과 나눗셈을 재미있게 학습할 수 있는 교육용 게임입니다.

### 주요 기능

- **3가지 게임 모드**
  - 곱셈 모드: 2~9단 곱셈 구구단
  - 나눗셈 모드: 기본 나눗셈 연산
  - 혼합 모드: 곱셈과 나눗셈 랜덤 출제

- **3단계 난이도**
  - 쉬움: 2, 3, 5단 (느린 속도)
  - 보통: 2~5단 (중간 속도)
  - 어려움: 2~9단 (빠른 속도)

- **게임 시스템**
  - 생명 시스템 (3개)
  - 콤보 시스템 (최대 3배 점수)
  - 최고 점수 기록
  - 학습 리포트 (정답률, 틀린 문제 분석)

## 🚀 시작하기

### 필요 조건

- Node.js 18.0 이상
- npm 또는 yarn

### 설치 및 실행

1. 의존성 설치
```bash
npm install
```

2. 개발 서버 실행
```bash
npm run dev
```

3. 브라우저에서 `http://localhost:5173` 접속

### 빌드

프로덕션 빌드 생성:
```bash
npm run build
```

빌드 결과 미리보기:
```bash
npm run preview
```

## 🎮 게임 방법

1. **게임 모드 선택**: 곱셈, 나눗셈, 혼합 중 선택
2. **난이도 선택**: 쉬움, 보통, 어려움 중 선택
3. **게임 시작**: 화면 상단에서 문제가 떨어집니다
4. **답 입력**: 키보드로 답을 입력하고 엔터키 또는 제출 버튼 클릭
5. **점수 획득**: 정답을 맞추면 점수를 얻고, 연속으로 맞추면 콤보 보너스!
6. **생명 관리**: 문제가 바닥에 닿으면 생명이 줄어듭니다

### 콤보 시스템

- 3연속 정답: 1.5배 점수 (Good!)
- 5연속 정답: 2배 점수 (Great!)
- 8연속 정답: 2.5배 점수 (Excellent!)
- 10연속 이상: 3배 점수 (Amazing!)
