## 🔍 문제 진단 방법

### 1단계: 개발자 도구 열기

**Chrome/Edge/Firefox**:
- `F12` 키 누르기
- 또는 마우스 우클릭 → "검사" / "Inspect"

### 2단계: Console 탭 확인

다음 메시지가 보여야 합니다:
```
App rendered, gameState: menu
```

#### 정상인 경우:
✅ `App rendered, gameState: menu` 메시지 표시
✅ 에러 없음 (또는 경고만 있음)

#### 문제가 있는 경우:
❌ 빨간색 에러 메시지
❌ "Failed to load resource" 메시지
❌ "Cannot read property" 같은 에러

### 3단계: Network 탭 확인

1. **Network** 탭 클릭
2. 페이지 새로고침 (`F5`)
3. 다음 파일들이 성공적으로 로드되는지 확인:

✅ 정상 (200 상태):
- `index.html`
- `index-*.css` (CSS 파일)
- `index-*.js` (JavaScript 파일)

❌ 실패 (404 상태):
- 빨간색으로 표시된 파일
- 404 Not Found 에러

### 4단계: Elements 탭 확인

1. **Elements** 탭 클릭
2. `<div id="root">` 찾기
3. 내부에 `<div class="App">` 있는지 확인
4. 그 안에 `<div class="main-menu">` 있는지 확인

#### 정상인 경우:
```html
<div id="root">
  <div class="App">
    <div class="main-menu">
      <h1 class="game-title">🌧️ 수학 산성비 게임 ⚡</h1>
      ...
    </div>
  </div>
</div>
```

#### 문제가 있는 경우:
```html
<div id="root"></div>  <!-- 비어있음 -->
```

## 🛠️ 해결 방법

### 방법 1: 브라우저 캐시 완전 삭제 (가장 중요!)

**Chrome/Edge**:
1. `Ctrl + Shift + Delete`
2. "전체 기간" 선택
3. 다음 항목 체크:
   - ✅ 쿠키 및 기타 사이트 데이터
   - ✅ 캐시된 이미지 및 파일
4. "데이터 삭제" 클릭
5. **브라우저 완전히 종료**
6. 브라우저 다시 열기
7. 게임 URL 접속

### 방법 2: 시크릿/프라이빗 모드 (강력 추천!)

**Chrome/Edge**:
```
Ctrl + Shift + N
```

**Firefox**:
```
Ctrl + Shift + P
```

시크릿 창에서 게임 URL 접속:
```
https://oooalways1.github.io/25110601/
```

### 방법 3: 강력 새로고침

게임 페이지에서:
- **Windows**: `Ctrl + Shift + R` 또는 `Ctrl + F5`
- **Mac**: `Cmd + Shift + R`

여러 번 시도하세요 (3-5번)

### 방법 4: 다른 브라우저

- Chrome

### 방법 5: 모바일에서 테스트
