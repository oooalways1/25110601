## 🆘 문제 해결

### 404 오류가 나는 경우
1. GitHub Pages 설정에서 gh-pages 브랜치 선택 확인
2. 몇 분 기다린 후 재시도
3. 브라우저 캐시 삭제 (Ctrl + Shift + Delete)

### 게임이 제대로 작동하지 않는 경우
1. 브라우저 콘솔 확인 (F12)
2. 최신 브라우저 사용 (Chrome, Edge, Firefox)
3. 페이지 새로고침 (Ctrl + F5)
